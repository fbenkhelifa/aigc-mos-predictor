# AGIQA-3K data acquisition notes

Use this checklist to keep dataset handling reproducible and compliant:

1. Identify the official AGIQA-3K release page and license terms.
2. Download raw files into `data/raw/agiqa3k/`.
3. Keep a local `SOURCE_NOTES.md` with URLs/date/checksum.
4. Run preprocessing scripts to generate `data/processed/` artifacts.

> Do not commit restricted raw assets unless redistribution is explicitly allowed.
